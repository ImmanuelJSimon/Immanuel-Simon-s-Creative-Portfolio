BoneBlitz Trivia Game

Objective:

In developing 'BoneBlitz Trivia,' my objective was to create an engaging and interactive web-based game that challenges users' knowledge of skeletal anatomy. The game presents players with images of different bones and asks them to correctly identify each bone from a set of randomized options. Through the process of designing and coding this game, I honed my skills in HTML, CSS, and JavaScript, gaining valuable experience in creating dynamic and responsive web applications. This project allowed me to strengthen my proficiency in user interface design, image manipulation, and algorithm implementation. Additionally, by incorporating features like randomized answer choices and dynamically loading images, I deepened my understanding of JavaScript programming concepts. 'BoneBlitz Trivia' serves as a testament to my ability to blend creativity with technical expertise to produce an educational and enjoyable online experience.
